Data Cleaning Process

The data cleaning process was done on Microsoft power query, The dataset was pulled into the power query using Get data From Range/CSV on Data Tab of Microsoft Excel; and the following steps were taken to clean the data.

Badly-structured-sales-Data-With-Analysis
  • Cancel/remove the "change type" carried out by the power query
  • Highlight column1 and transposed the table
  • Fill Down column1
  • Filter out 'Consumer Total', 'Corporate Total' and 'Home Office Total' on Column1
  • Promote first row as header
  • Highlight 'Column1' and 'Column2', then unpivot other columns
  • Depending on the version of Excel you are using, delete/remove the 'Order ID' containing Null values
  • Rename the column titles based on the sample cleaned data
2-Badly-structured-sales-Data-2
  • Cancel/remove the 'change type' carried out by the power query
  • Highlight column1 and trim the column
  • Highlight column1 and transposed the table
  • Fill Down column1
  • Promote first row as header
  • Highlight Column1 and Column2, then unpivot other columns
  • Split 'Attribute' column by delimeter
  • Remove 'Attribute.2' and 'Attribute.3' columns
  • Change the data type for 'Attribute.1' to date format
  • Rename columns to match sample cleaned data
  • Re-order Columns to start with 'Order Date'
3-Badly-structured-sales-Data-3
  • Cancel/remove the "change type" carried out by the power query
  • Merged 'Column 1' and 'Colmn 2'
  • Highlight merged column and transposed the table
  • Fill Down 'column1'
  • Promote first row as header
  • Highlight 'Column1' and 'Column2', then unpivot other columns
  • Split 'Attribute' column by delimeter {=}
  • Split 'Attribute.2' column by delimeter
  • Remove 'Attribute.2.2' and 'Attribute.2.3' columns
  • Change the data type for 'Attribute2.1' to date format
  • Rename columns to match sample cleaned data
  • Re-order Columns to start with 'Order Date'
5-Jumbled-up-Customer Details
  • Replace 'Name' with a delimeter {&}
  • Replace 'Address' with a delimeter{-}
  • Replace 'Age' with adelimeter {;}
  • Replace 'Gender' with a delimeter {=}
  • Split column by last delimeter {=}
  • Split column by next delimeter {;
  • Split column by delimeter (-)
  • Split column by delimeter {&}
  • Remove empty first column
  • Rename columns to match sample cleaned data
  • Change the data type for each column to match sample cleaned file
